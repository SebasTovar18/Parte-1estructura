1. Documentación Científica

Definición: Un Array o Arreglo es una estructura de datos indexada que almacena un conjunto de elementos homogéneos (del mismo tipo) de forma contigua en la memoria ram bajo un único nombre de variable.


Utilidad: Es la base conceptual para organizar colecciones fijas de datos donde se requiere un acceso inmediato y aleatorio a cualquier posición por medio de su índice.


Características clave:

  • Tamaño estático: Su tamaño debe definirse al momento de la compilación o instanciación y no se puede modificar en tiempo de ejecución.
  • Acceso directo (O(1)): Acceder a cualquier elemento toma la misma cantidad de tiempo si se conoce el índice matemático.
  • Homogeneidad: No permite mezclar tipos de datos diferentes en el mismo contenedor (ej. enteros con cadenas).

Casos de Uso reales: Gestión de los meses del año, almacenamiento de una matriz matemática de píxeles para procesamiento de imágenes, o buffers temporales de lectura de datos.

2. Ejemplos Prácticos Solucionados

Ejemplo 1: Declaración, llenado y lectura iterativa básica

El siguiente bloque en Java muestra cómo inicializar valores numéricos directos y leerlos mediante un ciclo clásico:

int[] numeros = {10, 20, 30, 40, 50};

for (int i = 0; i < numeros.length; i++) {
    System.out.println("Índice [" + i + "] -> Valor almacenado: " + numeros[i]);
}

Ejemplo 2: Arreglos de Cadenas de Texto (Objetos)

Muestra del manejo básico de vectores de tipo de dato referenciado String:

String[] videojuegos = {"Skyrim", "Call of Duty", "Apex Legends"};
System.out.println("El juego principal seleccionado es: " + videojuegos[0]);
