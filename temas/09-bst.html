1. Documentación Científica

Definición

Un Árbol Binario de Búsqueda (BST) es una estructura jerárquica donde los valores menores se almacenan a la izquierda y los mayores a la derecha.

Utilidad

Permite realizar búsquedas, inserciones y eliminaciones de manera eficiente manteniendo los datos ordenados.

Características Clave

  • Organización jerárquica.
  • Datos ordenados automáticamente.
  • Búsquedas rápidas.
  • Máximo dos hijos por nodo.
  • Recorridos ordenados.

Casos de Uso

Bases de datos, sistemas de búsqueda, gestión de información, índices y procesamiento de datos.

2. Ejemplos Prácticos Solucionados

Ejemplo 1: Crear nodo BST

class Nodo{

    int dato;

    Nodo izquierda;
    Nodo derecha;
}

Nodo básico BST.

Ejemplo 2: Crear raíz

Nodo raiz = new Nodo();

raiz.dato = 50;

Creación del nodo principal.

Ejemplo 3: Insertar menor

raiz.izquierda = new Nodo();

raiz.izquierda.dato = 30;

Valor menor a la izquierda.

Ejemplo 4: Insertar mayor

raiz.derecha = new Nodo();

raiz.derecha.dato = 80;

Valor mayor a la derecha.

Ejemplo 5: Mostrar raíz

System.out.println(
raiz.dato
);

Mostrar nodo principal.

Ejemplo 6: Mostrar izquierda

System.out.println(
raiz.izquierda.dato
);

Mostrar hijo izquierdo.

Ejemplo 7: Mostrar derecha

System.out.println(
raiz.derecha.dato
);

Mostrar hijo derecho.
