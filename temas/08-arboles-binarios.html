1. Documentación Científica

Definición

Un árbol binario es una estructura de datos jerárquica compuesta por nodos conectados, donde cada nodo puede tener máximo dos hijos: izquierdo y derecho.

Utilidad

Permite organizar información de manera jerárquica, facilitando búsquedas, recorridos y clasificación de datos.

Características Clave

  • Estructura jerárquica.
  • Cada nodo tiene hasta 2 hijos.
  • Existe un nodo raíz.
  • Permite recorridos ordenados.
  • Facilita búsquedas eficientes.

Casos de Uso

Sistemas de archivos, bases de datos, compiladores, motores de búsqueda y representación jerárquica.

2. Ejemplos Prácticos Solucionados

Ejemplo 1: Crear nodo

class Nodo{

    int dato;

    Nodo izquierda;
    Nodo derecha;
}

Nodo básico de árbol binario.

Ejemplo 2: Crear raíz

Nodo raiz = new Nodo();

raiz.dato = 10;

Creación del nodo raíz.

Ejemplo 3: Agregar hijo izquierdo

raiz.izquierda = new Nodo();

Inserción de hijo izquierdo.

Ejemplo 4: Agregar hijo derecho

raiz.derecha = new Nodo();

Inserción de hijo derecho.

Ejemplo 5: Asignar valores

raiz.izquierda.dato = 5;

raiz.derecha.dato = 20;

Asignación de datos.

Ejemplo 6: Mostrar raíz

System.out.println(
raiz.dato
);

Visualización del nodo principal.

Ejemplo 7: Mostrar hijo izquierdo

System.out.println(
raiz.izquierda.dato
);

Acceso al hijo izquierdo.

Ejemplo 8: Mostrar hijo derecho

System.out.println(
raiz.derecha.dato
);

Acceso al hijo derecho.

Ejemplo 9: Recorrido simple

System.out.println("Recorrido");

Inicio de recorrido.

Ejemplo 10: Crear árbol pequeño

Nodo arbol = new Nodo();

Inicialización de árbol binario.
