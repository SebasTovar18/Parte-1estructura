Tema 11: Grafos

Estructuras de Datos No Lineales

← Volver al Índice General

1. Documentación Científica

Definición

Un grafo es una estructura de datos no lineal formada por vértices (nodos) y aristas (conexiones) que relacionan información.

Utilidad

Permite representar relaciones, conexiones y recorridos entre diferentes elementos de forma eficiente.

Características Clave

  • Estructura no lineal.
  • Formado por nodos y conexiones.
  • Puede ser dirigido o no dirigido.
  • Permite recorridos complejos.
  • Representa relaciones reales.

Casos de Uso

Redes sociales, Google Maps, sistemas de rutas, internet, videojuegos y redes informáticas.

2. Ejemplos Prácticos Solucionados

Ejemplo 1: Crear nodo

class Nodo{

    String nombre;
}

Nodo básico de grafo.

Ejemplo 2: Crear vértices

Nodo A = new Nodo();

Nodo B = new Nodo();

Creación de vértices.

Ejemplo 3: Conectar nodos

System.out.println(
"A conectado con B"
);

Representación de conexión.

Ejemplo 4: Grafo dirigido

A → B

Conexión con dirección.

Ejemplo 5: Grafo no dirigido

A ↔ B

Conexión doble.

Ejemplo 6: Mostrar nodo

System.out.println(
"A"
);

Visualización simple.

Ejemplo 7: Crear conexión múltiple

A → B → C

Conexión entre varios nodos.

Ejemplo 8: Contar nodos

int nodos = 3;

Cantidad de vértices.

Ejemplo 9: Mostrar arista

System.out.println(
"Arista creada"
);

Representación de conexión.

Ejemplo 10: Ruta simple

A → C

Recorrido básico.
