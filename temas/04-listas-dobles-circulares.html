1. Documentación Científica

Definición

Las listas dobles y circulares son estructuras de datos dinámicas formadas por nodos enlazados. Las listas dobles permiten recorrer datos hacia adelante y hacia atrás, mientras que las listas circulares conectan el último nodo con el primero.

Utilidad

Permiten gestionar información dinámica de forma eficiente, facilitando recorridos continuos, inserciones y eliminaciones.

Características Clave

  • Memoria dinámica.
  • Recorridos flexibles.
  • Inserciones eficientes.
  • No requieren memoria contigua.
  • Permiten navegación bidireccional.

Casos de Uso

Playlists musicales, sistemas operativos, historiales de navegación, turnos rotativos y aplicaciones multimedia.

2. Ejemplos Prácticos Solucionados

Ejemplo 1: Nodo doble

class Nodo{

    int dato;

    Nodo anterior;
    Nodo siguiente;
}

Nodo con referencias dobles.

Ejemplo 2: Crear nodos

Nodo n1 = new Nodo();
Nodo n2 = new Nodo();

Creación básica de nodos.

Ejemplo 3: Enlazar nodos

n1.siguiente = n2;
n2.anterior = n1;

Conexión entre nodos.

Ejemplo 4: Lista circular

n2.siguiente = n1;

El último nodo apunta al primero.

Ejemplo 5: Recorrido

Nodo actual = n1;

while(actual != null){

    System.out.println(actual.dato);

    actual = actual.siguiente;
}

Recorrido secuencial.

Ejemplo 6: Insertar nodo

Nodo nuevo = new Nodo();

nuevo.siguiente = n1;

Inserción al inicio.

Ejemplo 7: Eliminar nodo

n1.siguiente = n2.siguiente;

Eliminación simple.

Ejemplo 8: Mostrar anterior

System.out.println(
n2.anterior.dato
);

Acceso bidireccional.

Ejemplo 9: Contar nodos

int contador = 0;

Inicialización para conteo.

Ejemplo 10: Turnos rotativos

actual = actual.siguiente;

Movimiento circular entre nodos.
